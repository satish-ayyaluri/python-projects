# 🧠 Quizzler App (Python)

## 📖 Project Overview

Quizzler is a desktop quiz application built using Python and Tkinter. It presents True/False questions, tracks the user's score, and provides instant feedback after each answer.

This project demonstrates Object-Oriented Programming (OOP), GUI development, and API/data handling concepts.

---

## 🚀 Features

- Interactive graphical user interface (Tkinter)
- True / False quiz
- Score tracking
- Instant answer feedback
- Displays final score after quiz completion
- Object-Oriented Python design

---

## 🛠 Technologies Used

- Python
- Tkinter
- Object-Oriented Programming (OOP)

---

## 📂 Project Structure

```
main.py              # Starts the application
question_model.py    # Question class
quiz_brain.py        # Quiz logic
ui.py                # Tkinter GUI
data.py              # Quiz questions
```

---

## ▶️ How to Run

1. Clone the repository

```
git clone https://github.com/satish-ayyaluri/python-quizzler-app.git
```

2. Navigate to the project folder

```
cd python-quizzler-app
```

3. Run the application

```
python main.py
```
