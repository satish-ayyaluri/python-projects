# PyPass Manager

A simple Password Manager built with Python and Tkinter.

## Features

* Generate strong random passwords
* Copy generated passwords to the clipboard automatically
* Save website, email/username, and password information
* User-friendly graphical interface using Tkinter
* Data stored locally in a text file

## Technologies Used

* Python
* Tkinter
* Pyperclip
* Random

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/pypass-manager.git
```

2. Navigate to the project folder:

```bash
cd pypass-manager
```

3. Install dependencies:

```bash
pip install pyperclip
```

4. Run the application:

```bash
python main.py
```
