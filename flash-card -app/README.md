🃏 French Flash Card App

A Python-based flash card application built with Tkinter and Pandas to help users learn French vocabulary. The application displays French words, automatically flips the card after a few seconds to reveal the English translation, and tracks learning progress by saving known words.

🚀 Features
Display random French vocabulary cards
Automatically flip cards to reveal English translations
Mark words as learned
Save learning progress using CSV files
Resume learning from where you left off
Simple and interactive Tkinter GUI
🛠 Technologies Used
Python
Tkinter
Pandas
CSV File Handling
Exception Handling
